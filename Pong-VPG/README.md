# Pong-VPG
Impelementation of a vanilla policy gradient to play game of Pong. Same idea as here https://spinningup.openai.com/en/latest/algorithms/vpg.html, but i estimate the value-function with monte-carlo method. I wanted to test how convolutional nets work with policy gradients.
